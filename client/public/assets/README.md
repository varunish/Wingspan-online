# Wingspan Assets

This folder contains visual assets for the Wingspan game.

## Asset Structure:

### Food Tokens
- `food/invertebrate.svg`
- `food/seed.svg`
- `food/fish.svg`
- `food/fruit.svg`
- `food/rodent.svg`

### Eggs
- `eggs/egg.svg`

### Dice
- `dice/invertebrate.svg`
- `dice/seed.svg`
- `dice/fish.svg`
- `dice/fruit.svg`
- `dice/rodent.svg`

### Board Elements
- `board/player-mat.jpg` - Player board background
- `board/action-cube.svg` - Action cube token

### Cards
- `cards/bird-back.jpg` - Back of bird cards
- Bird card images can be added as `cards/bird-{id}.jpg`

## Note:
For now, we're using CSS-based visuals. Replace these with actual game assets when available.
